# team/handoffs

Formal handoff packets between agents. Each handoff names the sender, recipient, readiness state, and next actions. Filename pattern: `YYYY-MM-DD-<from>-to-<to>-<slug>.md`.

## Conventions

- Filename: `YYYY-MM-DD-<slug>.md` for dated artifacts.
- Sign every entry with `— <AgentName>`.
- When an artifact is promoted to canonical docs, leave a stub here with a redirect link.
